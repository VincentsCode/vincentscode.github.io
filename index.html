WHAT I HAVE DONE SO FAR


THE SHOPPING-ROBOT

The conecept of the Shopping-Robot was to build a robot, which follows you, while you are walking through the streets or the shopping mall, so you could easily throw your purchases into an integrated compartment.

The idea was implemented using LEGO as a mechanical basis and Java/C++ as a software-environment. To track the users movement, a second Generation Microsoft Kinect Sensor was used.

THE HOMEWORK-BOT (WIP)

The Homework-Bot is basically an AI-Server which you can send a picture of some kind of homework to (currently only math is supported) and it will and it will solve the task and send it back to you. All of that is done by using neural nets to understand the task and the wolframalpha-api to solve equations if needed.
At the moment the project is still in really early stages, since its really hard to get enough math-tasks with solutions, to train a neural network.
